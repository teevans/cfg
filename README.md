## Helpful Info

### Neovim Python Provider Install

Python2
```bash
curl https://bootstrap.pypa.io/pip/2.7/get-pip.py -o get-pip.py
python -m pip install --user --upgrade pynvim
```

Python3
```bash
python3 -m pip install --user --upgrade pynvim
```
